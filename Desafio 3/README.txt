Para executar o código da maneira correta siga os passos a seguir:

1-Entre na pasta Desafio 3
2-Entre na pasta API
3-Execute o código nomeado de __init__.py em um servidor dedicado
4-Agora execute o código nomeado de sistema.py

Seguindo esse passo a passo será possível utilizar as funções disponíveis
no sistema como as funções: 
adicionar personagem
ver personagens
deletar personagens
ver o link do personagem.
